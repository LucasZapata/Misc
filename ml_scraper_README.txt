El programa acepta un texto que usa para la busqueda y los siguiente argumentos:
-url: toma el texto como url de partida en lugar de como término de busqueda
-depth: toma el número de páginas de resultados a leer
-p_min: toma un numero como precio mínimo
-p_max: toma un numero como precio máximo
-area: toma texto como area para la busqueda

El resultado se escribe en 'output.csv'
